Physics Engine

For as long as I can remember, I've been fascinated by the general theory of relativity, differential geometry, and the concept of higher-dimensional spaces. Inspired by ideas like those in the book Flatland, I've set out to build my own custom physics engine, one that can simulate and visualize different geometric manifolds governed by unique physical laws. It's an abitious and long-term project, and this page serves as a journal where I share my ideas, progress, and discoveries along the way.

GitHub Repository

1. Single Particle Under Gravity

Every journey begins with something small, and I choose to start this project by simulating a single particle under Earth's gravity. Using the semi-implicit Euler method with a one-second time step, the simulation illustrates how the particle's position evolves over a 10-second period. The program is also capable of handling multiple particles by utilizing an array of pointers to store each particle's state - including its position, velocity, and mass.
